## ✨ Features

* **5 Core Operations:** Supports addition, subtraction, multiplication, division, and modulo operations.
* **Smart Logic:** Multi-step operations chains smoothly without breaking.
* **Edge-Case Handling:** Prevents crashing or breaking when dividing by zero.
* **Clean UI:** Minimalist, user-friendly interface with responsive button animations.

## 🛠️ Built With

* **HTML5:** Semantic structure for the layout.
* **CSS3:** Custom styling, grid/flexbox layout, and active state transitions.
* **JavaScript (ES6):** Core calculation logic, event listeners, and state management.

## 🚀 Installation & Getting Started

Follow these simple steps to get the project up and running on your local machine.

### 📋 Prerequisites
You only need a modern web browser (Chrome, Edge, Safari, or Firefox) and Git installed on your computer.

### 🔧 Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the project folder**
   ```bash
   cd odin-calculator-project
   ```

3. **Launch the application**
   * **The Quick Way:** Double-click the `index.html` file inside the folder to open it directly in your browser.
   * **The Recommended Way:** Open the folder in [VS Code](https://visualstudio.com), install the **Live Server** extension, right-click `index.html`, and select **Open with Live Server**.
